# **App Name**: AyurWell

## Core Features:

- Symptom Analysis: Collect user-reported symptoms through a questionnaire to identify potential health imbalances.
- Dosha Identification: Analyze the user's symptoms using an AI tool and determine the primary Ayurvedic dosha imbalance (Vata, Pitta, or Kapha).
- Remedy Suggestion: Based on the identified dosha imbalance, the app will use AI to suggest natural home remedies, Ayurvedic treatments, herbs, yoga poses, and dietary tips. These recommendations are based on traditional Ayurvedic practices.
- Treatment tracking: Allow the user to track remedies, mark which they've used, and if they were helpful
- User Profile: Allow a user to save their personal info such as height and weight to customize recommendations.

## Style Guidelines:

- Primary color: Earthy orange (#C47944) to evoke nature and healing.
- Background color: Soft beige (#F5F5DC) to provide a calm and natural backdrop.
- Accent color: Muted green (#8FBC8F) to symbolize health and balance.
- Body and headline font: 'Alegreya', a serif font to give the feeling of tradition and intelligence.
- Use line icons representing various herbs, yoga poses, and Ayurvedic symbols.
- Clean and minimalist layout with a focus on readability and easy navigation.
- Subtle transitions and animations to provide a smooth and engaging user experience.
